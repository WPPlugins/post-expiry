== Description ==

Simple plugin to move your blog posts into another category at a specified time.  This plugin is useful for if you have a post that you want moved out of a featured category or one you want to archive without having to delete.  

Please feel free to contact me with bug reports or feature requests.

== Installation ==

1. Upload `post-expiry.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. A "Post Expiry" section will appear in your right hand sidebar when you add or edit a post.

== Frequently Asked Questions ==

= Do I have to set an expiry date for every post? =

No. The post will only be moved if a date is set.

= What if this plugin does not work? =

Please contact me through [tonibarrett.com](http://www.tonibarrett.com tonibarrett.com) and I'll look into the issue.
